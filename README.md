# RIs-atv

RI1
Este código cria personagens e compara seus atributos. Para usar, primeiro crie um personagem com a função criarpersonagem(), informando nome, codinome, armas e atributos como velocidade, força e resistência.

Os personagens são guardados em listas (arrays). Depois, todos são reunidos em todosOsPersonagens.

Quando o código roda, ele compara cada personagem com os outros usando Velocidade, Força e Resistência, mostrando no console quem vence em cada atributo.

Para adicionar um novo personagem, basta criar outro com criarpersonagem() e colocá-lo na lista.



RI2
Para usar o sistema, primeiro crie as informações de telefone e endereço. O telefone deve conter o DDD e o número, e o endereço deve conter rua, número, cidade e estado.

Depois disso, crie um cliente informando o nome e passando o telefone e o endereço criados. Cada cliente armazenará essas informações.

Se houver vários clientes, eles podem ser colocados em uma lista (array). Em seguida, utilize a função de ordenação para organizar os clientes em ordem alfabética pelo nome.

Por fim, utilize o método de exibição para mostrar os dados dos clientes no console.



RI3
Este código cria um sistema simples de cadastro de empresa e clientes, organizando informações como nome, 

CPF/CNPJ, telefones e endereços, e exibindo esses dados no console.

Primeiro, crie a empresa informando nome, CNPJ e endereço usando new Empresa().

Depois, adicione telefones da empresa com addTel().

Em seguida, crie clientes com new Cliente(), adicione dois telefones com addTel() e coloque os clientes na empresa usando addCliente().

Por fim, execute empresa.descricao() para mostrar no console os dados da empresa e dos clientes.

RI5
Este código cria um sistema simples em JavaScript para representar uma empresa, seus clientes, 

telefones e endereços. Ele permite cadastrar essas informações dentro das classes e armazená-las em listas.



No final, o método imprimir() mostra no terminal os dados da empresa, seus telefones e os clientes cadastrados.
